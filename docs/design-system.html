Design System · v0.2

Portal de Memórias

Aplicação Material colorida sobre superfícies tonalizadas. Esta versão consolida tokens, componentes e as telas principais num único documento navegável.

Parte 1

Fundamentos

Os tokens que sustentam a aparência do app. Tudo no resto do documento lê daqui — não se inventa cor, tamanho, peso ou tempo fora desta lista.

1.1 · Cores

Cinco acentos cheios (Google + roxo) + paleta de superfícies tonalizadas. O fundo principal é levemente tonalizado, não branco puro — o branco fica reservado para cards e modais.

Parte 5

Vocabulário

Convenções de escrita e formatação que toda tela deve respeitar.

5.1 · Tratamento de data

Sempre em PT-BR. Nunca usar formatos numéricos (03/14/2026) em UI visível — só em tooltips ou campos técnicos.

OndeFormatoExemplo
Cabeçalho de mês (Linha do tempo / Calendário)Mês de AnoMarço de 2026
Data principal de uma fotoDia de mês14 de março
Quando o ano importaDia de mês de ano14 de março de 2026
Adicionado recentementeRelativohá 2 dias
Tooltip / campo técnicoNumérico2026-03-14 18:32

5.3 · Telas que faltam pra v0.3

Esta v0.2 cobre o essencial. Próxima versão (v0.3) deve incluir: Login, Mapa, Lista, Lixeira, Modal de upload em lote, Dialog de confirmação, Dropdown menu, Loading skeleton, Date picker, Combobox autocomplete.
